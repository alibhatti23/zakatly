# Zakatly

A free, accurate, ad-free Zakat calculator. Open source under the MIT license.

## Feature roadmap

The build follows a tiered roadmap:

### Beginner (core essentials)

- Nisab setup (gold/silver rate, currency)
- Cash and bank
- Gold and silver
- Result and summary
- Fiqh standard toggle

### Intermediate (full picture)

- Investments, business assets, liabilities, real estate
- Info tooltips, export as PDF, multi-language, live rate API

### Advanced (power features)

- Agriculture, livestock, lunar reminder, history log
- Multi-currency, public API, offline PWA, contributor docs

## Development

```bash
npm install
npm run dev
```

## Tech stack

React, TypeScript, Vite, Tailwind CSS.

## Contributing

Each feature is built on its own branch and merged through a pull request. See open PRs for work in progress.
